# CSE15L Tutorial 

Welcome to my tutorial for new CSE15L students. \
This guide will include step-by-step instructions to install VSCode, remote connect to ieng6, and try some commands.

## Installing VSCode
Feel free to skip this step if you are not planning to use your personal laptop for any of the labs in this course (which you don't have to).
1. First, go to [the Visual Studio Code website](https://code.visualstudio.com) <img width="1424" alt="image" src="https://user-images.githubusercontent.com/122579697/212215180-e252bc26-f9f2-4d35-b746-3297822ff28b.png">

2. Next, click the blue icon that says "download" in the upper right-hand corner. <img width="1425" alt="image" src="https://user-images.githubusercontent.com/122579697/212215137-ac6e3a35-5aca-4d05-87cc-6463a8c3aa94.png">

3. And then choose the operating system that you are using (macOS, Windows, or Linux) and click it. 
4. Now open the file you just downloaded and start installing
5. After successfully installing VSCode, you should have something like this after opening it: <img width="1024" alt="image" src="https://user-images.githubusercontent.com/122579697/212215332-9619e1e3-1c4a-4bdc-9c9d-03361aaf6f79.png">
